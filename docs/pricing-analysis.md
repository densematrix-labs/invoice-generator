# Pricing Analysis

## Competitor Snapshot

Invoice-generator.com is freemium: single invoices can be created quickly, while user accounts, document history, templates, and business workflow features create the upgrade surface.

## Cost Model

- Core document creation: near-zero marginal cost.
- PDF export: browser print, near-zero marginal cost.
- Server cost: shared demo host.
- Payment processing: Creem MoR fees estimated at 5-6%.
- Support risk: low, because no accounting or tax advice is provided.

## Pricing Strategy

| Tier | Price | Target User | Included |
|------|-------|-------------|----------|
| Free | $0 | One-off invoices | Unlimited local drafts, print/PDF export, DenseMatrix footer |
| Starter | $5/month | Freelancers | Saved templates, 25 client records, branding removal |
| Studio | $12/month | Solo operators | Recurring invoices, payment links, client history |
| Business | $29/month | Small teams | Team templates, bulk client import, priority export history |

## Paid Hooks

- Save templates
- Recurring invoices
- Payment links
- Branding removal
- Client history
- Export archive

## SKU List

- `starter_monthly`: $5/month
- `studio_monthly`: $12/month
- `business_monthly`: $29/month
